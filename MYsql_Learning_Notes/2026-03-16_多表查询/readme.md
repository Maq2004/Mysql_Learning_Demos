内连接 vs 外连接核心区别

内连接：只保留“双方都匹配上的记录”（不留情面，无 NULL）
左外连接：左表全部保留，右表不匹配的填 NULL
右外连接：右表全部保留，左表不匹配的填 NULL

右边独有（B - A∩B）：左表对应不上 → RIGHT JOIN + WHERE 左表字段 IS NULL
