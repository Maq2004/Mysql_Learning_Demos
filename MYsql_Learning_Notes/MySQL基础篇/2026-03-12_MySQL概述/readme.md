## 回顾10问（数据库概述核心问题）

1. **为什么要使用数据库？**  
   持久化（把内存数据保存到可掉电设备）。文件系统（如Excel）在数据量大时容易崩溃，而数据库能解决持久化、共享、一致性问题。

2. **DB、DBMS、SQL分别是什么？三者关系？**  
   DB = 仓库（文件系统）  
   DBMS = 管理软件（建立、使用、维护数据库）  
   SQL = 通信语言（专门与数据库对话）  
   一个DBMS可管理多个DB，一个DB可有多张表。

3. **一个DBMS可以管理多少个数据库？一个数据库里可以有多少张表？**  
   一个DBMS可以管理多个数据库；一个数据库可以有多张表（表用来存实体数据）。

4. **目前最受欢迎的DBMS前5名是什么？**  
   1. Oracle  
   2. MySQL  
   3. Microsoft SQL Server  
   4. PostgreSQL  
   5. MongoDB（前4个是关系型，MongoDB是文档型）

5. **MySQL发展史重大事件？**  
   2008年被Sun收购，2009年Sun被Oracle收购（间接拥有MySQL）。MariaDB因担心闭源风险而诞生。

6. **为什么那么多公司选择MySQL？**  
   开放源码、性能卓越、体积小、使用简单、易维护、历史悠久、社区活跃、支持多种语言、能处理上千万条记录、经过时间验证。

7. **RDBMS和NoSQL的本质区别？**  
   RDBMS：行+列存储、SQL查询、事务支持、复杂查询强。  
   NoSQL：键值对/文档/列式/图形存储、不用SQL解析、性能高、功能阉割版、免费。

8. **表、记录、字段分别对应什么？**  
   表 ≈ 类  
   记录（一行） ≈ 对象  
   字段（一列） ≈ 属性  
   这就是ORM思想（对象关系映射）。

9. **表之间的关联关系有哪四种？**  
   一对一、一对多、多对多、自我引用。

10. **一对多和多对多建表原则是什么？**  
    一对多：在从表（多方）加外键指向主表主键。  
    多对多：必须建第三张中间表，中间表至少放两个外键，分别指向两个表的主键。
